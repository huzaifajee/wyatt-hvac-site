WYATT HVAC STATIC WEBSITE

Files:
- index.html
- assets/style.css
- assets/script.js

How to run:
1. Open index.html in your browser.
2. Or upload the complete folder to GitHub.
3. In GitHub repository: Settings > Pages > Deploy from a branch > main > /root.

Important:
- The contact form is frontend-only. Connect it to Formspree, EmailJS, Netlify Forms, or a custom backend.
- Background photos currently load from Unsplash. Download and store them locally before production if you want full control.
- Phone numbers, address, email, text and brand assets are placeholders and should be replaced.
